# KEMI FLY

케미가 하늘섬 사이를 날며 수정과 AI 구름을 모으는 모바일/웹용 원터치 비행 게임입니다.

## 실행

```bash
npm run dev
```

브라우저에서 `http://127.0.0.1:5173`을 엽니다.

## 조작

- 모바일/웹: 탭 또는 클릭으로 상승
- 키보드: `Space` 상승, `P` 일시정지, `M` 사운드 토글, `F` 전체화면, `Enter` 시작/재시작

## 기능

- 충돌 즉시 게임오버가 아닌 에너지 감소 방식
- 모바일 세로/가로 및 데스크톱 화면에 맞춰 캔버스와 HUD 자동 전환
- 초록/금색 `+` 링으로 표시되는 수집 아이템과 붉은 `!` 링으로 표시되는 장애물
- 수정 점수, AI 구름 회복, 보호막 아이템, 근접 회피 보너스, 콤보/피버 보상
- 아이템 수집 개수에 따라 Lv1~Lv6까지 케미 외형과 HUD가 성장
- 수정 수집, 근접 회피, 비행 거리 기반 미션 진행
- MP3 배경음악과 Web Audio 기반 효과음
- 메뉴 배경 이미지를 살짝 확대해 천천히 패닝하고 은은한 반짝임 연출
- 원거리 배경 루프 이동과 바람선으로 비행감 표현
- 진행 거리에 따라 장애물 속도가 빨라지고, 일정 거리 이후 용암 지대로 전환
- 먹는 아이템은 밝은 보상 아이콘과 초록/금색 `+` 링, 장애물은 돌/보라/용암 계열과 붉은 `!` 경고 링으로 구분
- 브라우저 localStorage 기반 명예의 전당 Top 10
- Vercel 배포 시 Vercel Blob 기반 명예의 전당 Top 10 영속 저장
- `manifest.webmanifest`와 서비스 워커 기반 설치형 웹앱 준비
- Playwright 검증용 `window.render_game_to_text`, `window.advanceTime(ms)` 제공

## 보안 및 데이터

- 배포 환경에서는 같은 출처의 `/api/leaderboard`만 호출합니다.
- 외부 오디오/이미지 CDN 없이 로컬 에셋, 로컬 MP3/MP4, 브라우저 Web Audio만 사용합니다.
- 입력 이름은 클라이언트에서 길이와 문자 범위를 제한하고, 화면 출력 시 DOM 삽입 대신 Canvas 텍스트로만 렌더링합니다.
- 명예의 전당 기록은 서버 API에서 다시 검증한 뒤 Vercel Blob에 저장합니다.
- Blob 저장소가 준비되지 않은 로컬 개발 환경에서는 기존 localStorage로 자동 폴백합니다.
- 개발 서버는 `127.0.0.1`에만 바인딩합니다.
- 배포 시 `BLOB_READ_WRITE_TOKEN`은 Vercel 환경변수로만 설정하고 저장소에 커밋하지 않습니다.

## 배포 준비

1. Vercel 프로젝트에 Blob Store를 연결합니다.
2. Vercel 환경변수 `BLOB_READ_WRITE_TOKEN`을 설정합니다.
3. GitHub 저장소와 Vercel 프로젝트를 연결한 뒤 Preview 배포부터 검증합니다.
